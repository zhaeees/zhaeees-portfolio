## 💡 Portfolio Focus

이 포트폴리오는 다음과 같은 목표를 중심으로 제작되었습니다.

1. 사용자 경험 (UX) 중심 설계

직관적인 UI와 자연스러운 인터랙션을 통해
사용자가 쉽게 이해하고 사용할 수 있도록 설계했습니다.

2. 반응형 웹 구현

다양한 디바이스 환경에서도 일관된 사용자 경험을 제공하기 위해
모바일 퍼스트 기반으로 반응형 구조를 설계했습니다.

3. 인터랙션 및 애니메이션

스크롤, 타이핑, 슬라이드 등 다양한 인터랙션을 활용하여
동적인 사용자 경험을 구현했습니다.
